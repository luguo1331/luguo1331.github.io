Hexo + GitHub Pages 搭建个人博客

发表于 2018-05-14 | 分类于 Hexo

Hexo 是一个快速、简洁且高效的博客框架。

本文将简单介绍如何利用 Hexo 和 GitHub Pages 搭建个人博客

安装前提

安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装下列应用程序：

  • Node.js
  • Git

如果您的电脑中已经安装上述必备程序，那么恭喜您！接下来只需要使用 npm 即可完成 Hexo 的安装。

npm install -g hexo-cli

建站

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

hexo init

npm install

写作

hexo new “文章名”

新建的文章将会存在 source/_posts 中，_posts 就是默认模板，文章文件名称将会是 文章名.md，然后打开这个文件编辑完保存就ok了。

生成静态文件

写完博客后生成静态网页文件

hexo g

本地调试

本地服务器启动完之后，访问 http://localhost:4000 来验证一下你写的文章预览效果。

hexo s —debug

部署到 GitHub

安装部署工具

npm install hexo-deployer-git —save

修改配置文件

打开主配置文件，填上你的仓库地址

deploy:
  type: git
  repository: https://github.com/xxx/xxxx.github.io.git
  branch: master

ssh这些不用讲了自己配一下

最后部署网站到 GitHub

hexo d

之后如果再发布文章，可以直接用以下命令

hexo clean
hexo d -g
